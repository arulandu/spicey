# Spicey
Spicey is a Rust DSL for circuits that supports interop with SPICE netlists.

## Roadmap
- [ ] MVP:
    - [x] Parsing and validation of basic circuit components
    - [x] Netlist generation from DSL
    - [ ] Calling NgSpice
    - [ ] Simulation
- [ ] Features:
    - [ ] Subcircuit support 
    - [ ] Transient and controlled sources
    - [ ] Models
